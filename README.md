# Angular Pipes.

## Что такое pipes?

- используют для отображения чего либо в шаблоне. Дополнительные обработчики входных данных.

## Какие кейсы применения pipes?

- значение под формат.
- decimal pipe.
- локализация.
- сортировать и фильтровать.
